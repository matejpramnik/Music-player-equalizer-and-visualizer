# **Music player, equalizer and visualizer**

This is a project for my bachelor's thesis. It's a music player for many popular audio files with a standard equalizer that also visualizes music as it's playing. There are currently 7 visualizations to choose from. The app currently supports two languages - English and Slovak.

## **Requirements:**
Python 3.13.12

FFmpeg 8.0.1 or newer

**To install all required python modules, use:** 
```
python -m pip install requirements.txt
```
